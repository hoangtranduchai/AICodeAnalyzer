# Roadmap

This roadmap lists practical improvements after the current academic-ready version.

## Near Term

- Add richer UI validation messages.
- Add visual progress timeline for crawl and analysis batches.
- Add report preview before export.
- Add user-selectable date range for reports.
- Add stronger retry controls for provider quota errors.
- Add more parser tests for Codeforces and VJudge page variants.

## Medium Term

- Add migration tool support.
- Add encrypted source storage option.
- Add installer packaging.
- Add UI automation tests.
- Add configurable report templates.
- Add source diff and history view.
- Add more online judge platforms.

## Long Term

- Add role-based multi-user mode.
- Add centralized job queue.
- Add cloud database option.
- Add dashboard analytics over time.
- Add rubric customization.
- Add model comparison and evaluation harness.
- Add privacy-preserving anonymized exports.

## Non-Goals

- Captcha bypass.
- Hidden source extraction.
- Final misconduct verdict automation.
- Storing browser cookies or platform tokens.
